# Stagecast Sign Up tool

## Technologies Used:
* HTML
* CSS
* JavaScript (React)

### Packages installed
* ` npm i react-router-dom`
* ` npm i react-router`
* ` npm i throttle-debounce`
* `npm install @material-ui/core`

# Functionality

* Users provide the required fields of the form
* They provide password
* The login is automatic
